【さくらVPS】服务器的搭建

0.选择[さくらVPS]

为了验证某些不可告人的技术，空出来我的树莓派3来做一些尝试，所以我决定将在家搭建的博客等网站转移到较稳定的公网服务器上。
部署网站其实可选的候补还是有很多的，比如AWS EC2，VPS，共享服务器。虽然是老生常谈的话题，但是还是有必要再聊一聊这几种部署方式的选择，为一些朋友提供一些帮助。请参照这篇文章。

2.申请[さくらVPS]

3.服务器所需要的软件

  • VIM - 编辑器
  • Docker - 必不可少，有了它基本上其他都不需要了。再次感叹一下，有了Docker的世界真美好。
  • Docker Compose -

4.服务器的搭建过程

4.1 更新

第一步自然是更新软件数据库了。

$ sudo apt-get update

4.2 安装vim

$ sudo apt-get install vim

4.3 安装Docker CE - 社区版

今年3月份随着Docker EE - 企业版的发布，原有的Docker分为了Docker EE和Docker CE，作为个人的我毫无疑问选择使用社区版。
官方版：Ubuntu下的Docker CE安装说明请参照 https://docs.docker.com/engine/installation/linux/docker-ce/ubuntu/

1.允许apt使用HTTPS协议的仓库。

$ sudo apt-get install \
      apt-transport-https \`
      ca-certificates \
      curl \
      software-properties-common`

2.添加Docker官方的GPG key。

$ curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

3.添加Docker的仓库。 

$ sudo add-apt-repository \
     "deb [arch=amd64] https://download.docker.com/linux/ubuntu \
     $(lsb_release -cs) \
     stable"

4.再次更新软件数据库。

$ sudo apt-get update

5.重头戏来了，终于要安装Docker CE了。

$ sudo apt-get install docker-ce

6.验证一下安装是否成功

$ sudo docker run hello-world

如果看到输出Hello from Docker!了，恭喜你，你成功了。
